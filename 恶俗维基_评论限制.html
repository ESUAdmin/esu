评论规则

  • 讨论与条目无关的内容（删除）
  • 发表与政治相关的内容（删除）具体标准和解释参见 恶俗维基:免责声明
  • 发表令人大脑降级的内容（删除）
  • 指点江山（封禁IP）
  • 在评论区脱粪（封禁IP段/全运营商封禁）
  • 多次违反评论规则（封禁IP段/全运营商封禁）
  • 使用代理服务器（数据中心）违反任意评论规则（全运营商封禁）

在发送评论时，如果提示“您的评论内容违反了评论规则”，则是您的评论包含被禁止的内容，请尝试修改评论内容后重试（不要包含不文明的言论、广告推广、不能理解的文本等内容）。

如果您多次尝试发送此类内容，您的 IP 地址可能会被系统自动封禁，您需要等待一会才能继续发表评论。

登录用户

登录用户的评论没有任何限制，一经发布可被立即显示。

未登录用户

评论缓存

未登录用户的评论存在缓存，在发布成功后需要等待几分钟才能正常显示，无需重复发送。

IP 封禁

在发送评论时，如果提示“没有权限发布评论”，则是当前使用的 IP 地址遭到封禁。

造成 IP 地址封禁的原因和解决办法如下：

  • 与您 IP 地址相近的人（很有可能就是你自己）在评论区大放黑屁。同时，因部分管理员使用范围封禁，也可能造成您的 IP 被连带封禁。
如果您确信自己从未进行过破坏活动，请尝试申诉以获得IP封锁例外。
  • 您的整个城市的 IP 地址被封禁导致您受到影响，这是因为您所在的城市存在大脑严重有问题的人，当它的 IP 被封禁后，它会移动到其他位置蹭其他人的 WiFi 进行持续性的破坏。
这些封禁会在一段时间后得到解除，您可以发送邮件进行查询。

运营商封禁

在发送评论时，如果提示 “无法连接到服务器”，则是当前使用的 IP 地址所在的网络自治域系统（ASN）遭到封禁，也就是整个运营商被封禁。

由于本站在部分地区遭到封锁，运营商封禁系统目前暂时处于停用状态。

造成运营商封禁的原因和解决办法如下：

  • 你正在数据中心访问本站，这通常是因为你正在使用公共代理（Open Proxy）或者虚拟专用网络（VPN）
请停止使用这些软件，然后重新尝试发送评论。解决 DNS 污染可使用 恶俗维基:地址更新器。
  • 大量与你同一运营商的人在评论区大放黑屁，并且该运营商所在的国家访问本站的人数较少。
你可以尝试更换运营商，或者移民。

申诉

  • Q: 需要对页面内容进行编辑
A: 成功申请账户并登录后即可对页面进行编辑，也可以无限制的发表评论。
  • Q: 需要对页面内容进行投诉或举报（例如存在虚假内容，侵犯版权等）
A: 通过本站首页上公布的联系方式（邮箱/Twitter）发送请求，请注意：所有请求都将被公示。
  • Q: 需要对条目的重要内容进行补充，但不想注册账户
A: 请与本站任何用户或首页上的邮箱取得联系。
  • Q: 需要发表评论
A: 请将评论内容写在 TXT 内，然后拖放到回收站里。
