一、什么是风评被害？
风评被害：一部MAD爆红之后，若是选用的素材的名气比不上MAD本身，而因此被取材的动画被观赏MAD的观眾吸引同时留言一部分被MAD相关的弹幕佔据（原本毫无关係），我们就称之為风评被害。
ニコニコ上着名的风评被害曲
やらないか（要来一发吗？）的改编曲（被害者「バラライカ」）
ゲッダン(被害者「Promise」)
通常NICO厨们听到这类风评被害曲子会先联想到MAD而非原曲；而对於淫梦民而言，只要一听到原曲就会响起淫梦版MAD的幻听，此时原曲动画评论充斥着淫梦版MAD的台词捏他。
一般的风评被害可能仅止於幻听，但淫梦系则是能以各种理由引伸、变成风评被害的因素：部分动画平白无故地被淫梦民入侵淫梦动画专用的弹幕，只因為该动画有部分内容跟淫梦有关联（这种关联性并不像上面的例子是使用歌曲穿插全部动画，可能只是影片中的一个小画面有与淫梦捏他稍微碰到）。
通常引起淫梦民兴趣的这种小捏他只有淫梦民知道，却常常高调地在无辜的原动画留下淫梦特有的弹幕，故常常招来一般观眾的不满，部分风评被害的相关百科被大量删除的留言大多数都是淫梦民的留言。
第一类：只因為样貌与野兽先辈或许在某个角度时会好像有一点点儿相像
伊藤かな恵（声优）
长友佑都（足球选手，淫梦民称為「国米长友（インテル长友）」，在特定画面必定会出现其弹幕）
西寺郷太（乐团nona reeves成员，PV中一个顏面转盘的画面常被用於BB先辈剧场动画中）
北岛康介（游泳国手，比起样貌来说与四章同為游泳部的关联比较大）
第二类：只因為剧情上可能有点儿会能引起淫梦民联想的地方
しゅわスパ大作戦☆（东方PV，萃香下药的画面与四章类似，萃香也因此被认為是野兽先辈的妹妹）
ゆるゆり（摇曳百合）（动画，同性恋爱的题材以及第五话中的出演其中压迫的情节以及一些小捏他与淫梦本编激似）
第三类：只因為名称可以被联想到
只要名字能跟淫梦捏他產生连结都有机会被淫梦民所入侵，以下举例：
真夏の夜の雪物语（R-18游戏）
The Beast.（VOCALOID原创曲）
卯月の嘘（VOCALOID原创曲，作者受此事件影响已退隐并删除所有作品）（为他默哀，同时也请淫梦厨克制下）
远野物语（书籍）
第四类：当事人单纯地玩一下淫梦捏他
例：悠木碧、早见沙织。
第五类：其他更荒谬的理由
淫梦关连动画编号：例如sm114514、sm364364、sm893810等等。虽然多数动画早就在风评被害前已经被删除，但仍因為评论机制仍存在因此被淫梦民攻陷（但事实上这类删除动画也没什麼被害的意义，只是淫梦民间的乐趣罢了）
まんが日本昔ばなし（动画，同样因為淫梦关联动画编号sm893893而风评被害，不过由於被害前未删除且為有名物，所以后来虽然被权利者删除却惹得整个系列被归入淫梦的分支动画裡，主标籤為日ぺ昔话）
菅野美穗（日本女演员和歌手，迫真空手部中入浴场景的意义不明话语的空耳）
二、风评被害的形式
淫梦风评被害的形成有几种形式，大抵上可以分成有意（特意发表动画或评论将非关联有名物跟淫梦捏他相结合藉以达到风评被害的目的）跟无意（恰好被淫梦本编作為素材、或者被选材做成淫梦MAD后爆红的素材為此类）两种。详细方式如下：
①联想赤字兄贵
最直接且广泛的风评被害方式。淫梦动画的习惯是在影片裡用置底大红字放捏他（？），部分赤字兄贵会把联想到的类似事物作為红字弹幕的内容，若共鸣性高可能会延伸至其他所有淫梦动画的相同场景，最后变成习惯。
例子族繁不及备载，像是红魔馆→立教大学、イ ン テ ル 长 友、野 田 内 阁、八 王 子、铃 木 福等等。
②特定兄贵
明确指示出淫梦动画选用素材的详细资讯的兄贵。一般动画不会特意做这种鉅细靡遗的描述，所以很明显就是為了要风评被害。
③问题シリーズ（问题系列）
如同字面所述，本系列是把某些与淫梦毫无关係的影片播放，遇到问题场景时再重复播放，最后放上关联淫梦影片作為模范解答，通常一播到问题场景（对话）时就已经被淫梦民猜到正解影片的内容了。
最早是[エア本动画>久本雅美]专用的系列，但是由於近来UNEI的赶尽杀绝，加上调性非常适合淫梦动画於是就转变為淫梦动画的系列之一。
部分动画甚至连常见的用句都被抓出来只因為是淫梦的名言因此被当成问题发言（像是そうだよ、わすれてた等等），不过其实エア本在这系列就是这样作的。
三、由风评被害到被侵蚀成為淫梦分支的系列
1、Nona Reeves
约20年前活跃的流行曲乐团，主要人物有西寺乡太（NSDR）及You The Rock★（YTR★），尚有其他成员但是在淫梦圈活跃的主要还是前面两者。
被牵连的元凶：歌う野獣先辈
原影片作者称在2:25时NSDR的表情被认為激似野兽先辈（一般认為有点牵强，但事实上还有比这个角度更像的镜头） 但是该动画仍然被淫梦民们视為圣地（地位近似於摔角系列的本格内裤摔角动画，衍伸不少特定格式的弹幕），NSDR甚至因此被直接当成野兽先辈的素材使用。
另一位客串演出的YTR★下场也不怎麼好，不过一部分跟现实的负面消息有多少影响就是了。
不过或许是风评被害的影响西寺本人由於再度受到关注也重新活跃起来，接受访问甚至办起Live。「月曜先辈」的称呼就是从本人口中说出，由此可见淫梦的影响力。
此乐团在淫梦民当中拥有极高的人气，深受淫梦民喜爱的主唱西寺郷太还有着「名曲しか作れない男」、「名曲製造机」等称号。
2、Ryu☆
知名音乐游戏曲子作者。跟淫梦扯上关联只因為「I'm so happy」接着由於该曲对淫梦的爆炸性影响，连带的让其他曲子也受到影响。
「I'm so happy」的走红源自於音MAD「霊梦 so クッキー☆」，其中这部音MAD将淫梦及クッキー☆连结，同时使用了「一转攻势」的手法，一般认為该动画对淫梦MAD產生了极為深远的影响。
「I'm so happy」引发流行后淫梦民开始对类似的歌曲下手，找上了sakura系列的曲子并製作「ﾌﾞ　ﾙ　ー　ﾍ　ﾞﾘ　ー　ﾈち　ゃ　ん」：其使用了Sakura storm的曲子，且仿照I'm so happy採用一转攻势，因此一样大流行。且為了「致敬」I'm so happy系列对元凶的编辑手法致敬的方式，其衍伸的MAD同样採用与sakura storm元凶动画一样的编辑方式。
此音MAD衍生出「ブルーベリーシリーズ（蓝莓/蓝色贝莉奴系列，使用ストライクウィッチーズ作為素材的一转攻势系列）」及「○ ○ s t o r m 系 列（用Sakura storm致敬元凶动画的系列）」。
接着「sakura reflection」、「sakura sunrise」同样复製了Sakura storm的模式，被害者依然是蓝莓。当然共通特色都是致敬元凶以及一转攻势。
这系列中选用的素材不见得必须包含淫梦要素，但是由於淫梦一转攻势的印象太过强烈，因此这类曲子的所有音MAD都无法避免淫梦弹幕的侵入。
3、クッキー☆
全（原）名「【东方合同动画企画】魔理沙とアリスのクッキーKiss」，是淫梦史上最大的风评被害。
动画内容為东方百合向的集体创作故事（企划主办人為莲奈理绪，作画、配乐、编集、配音等共计43名製作者），剧情大意是魔理沙製作情人节巧克力饼乾送给爱丽丝的过程，全长30餘分。
种种的粗劣製作理由让其在东方圈中评价不高，往后发生的各种事件让它演变成现时淫梦最大的分支。

1、ひと☆ごみ兄贵
クッキー☆及蓝莓系列堕落的最大元凶，同时带起一转攻势潮流的起始者。
由於其作品的在淫梦界的爆发性因此在クッキー☆第一次东方厨与淫梦厨的战争中成為牺牲者遭UNEI删除，但也阻止不了后继者的怒涛攻势。
编辑技巧普通，影像也只是用了简单的反转或扭曲特效，但也因為如此反而更容易使后继者仿效，產生了大量致敬原作的作品。
2、oh兄贵
动画以BB先辈剧场以及BBクッキー☆剧场系列為主，庞大的投稿密度却有一定水準。
ALC在BBクッキー☆剧场中被黑化的最大元兇。
SIK素材最大利用。
3、石（クッキー☆）
ケモ兄贵
在クッキー☆圈非常活跃，与非常多製作者关係友好，数次与oh兄贵同时投稿性质相近的BBクッキー☆剧场动画，即使题材不同，小捏他也会相互参考。
4、48兄贵
动画以纯クッキー☆要素出演的音MAD為主，偶有淫梦成分的作品。
从投稿动画中流露满满的对UDK姐贵的爱。以眾多UDK单独出演的音MAD為代表作，且相当直球的标题（选曲），成為UDKファンの圣地的代表。
5、朗読兄贵
以朗读的方式出演，通常作為旁白演出。
6、ほりお兄贵
以HIP HOP风格的クッキー☆及淫梦共演的音MAD為名，动画把淫梦角色塑造成饶舌歌手的形象為其特徵。
7、2号兄贵
动画以传统クッキー☆音MAD及淫梦音MAD為主。
擅长重新编曲、自创曲以及污物手绘人物/背景绘画，并将部分自作曲子作成音MAD，以带有哀愁感的Techno曲风加上颇具个性的剪辑及充满艺术感的画面编辑（其实是為了掩盖会被删除的问题画面所使用）获得不少人气。
知名作「Plasticookie☆Adventure.mp4」為目前クッキー☆系音MAD的顶点（非原创曲，被运营删除后再投稿仍达成30万再生）。
8、なで肩兄贵
动画以クッキー☆音MAD及淫梦音MAD為主，两边素材的使用上不受一转攻势等格式的拘束。
音乐及动画都有相当水準的音MAD作者，独创的「便乗ベース（便乘BASS）」為日后淫梦音MAD中常用的音源。
9、龙门渕ダマスカス兄贵
クッキー☆ MAD知名作者，作品不多但不论影像（基佬特有的60FPS）或者声音具有极高水準，非经济时段观赏推奖。
10、织部兄贵
加拿大在住兄贵，因此素材与选曲有其独特的SENSE。
品质堪称最高等级，其作品特色為作风清新、几乎都有一转攻势。
11、可能栄光（イケメン兄贵）
初期以RIM（RU）ONLY的音MAD走红。
投稿速度极快、品质亦高。素材运用以及影片呈现的方式具独创性。
12、のち身兄贵
长期以来一直在活动的稳健派製作者，一旦出手就是大作。
技术与内容并重，充分掌握クッキー☆内容、从中选用洽当的素材，令人印象深刻。
13、仲良子子吉兄贵
灵活运用所有クッキー☆、クッキー☆☆以及其餘所有素材的程度的能力（申し訳程度の东方要素）
熟知クッキー☆檯面上下的歷史、融合在音MAD中，要完全了解MAD中每处的捏他需要非常丰富的クッソ无駄な知识。
14、ALISON兄贵
投稿风格迥异的クッキー☆及淫梦MAD作者，有自己的形象图像并作為角色出演（图片取材来自於主催者莲奈理绪的クッキー☆脚本草稿中的RIM）。
动画内容包含中东、毒品、宗教等主题，一般对ALISON兄贵动画的感想為充满危险性、不安感。
15、VOX兄贵
具高超技巧的クッキー☆音MAD作者，虽然动画长度偏短极具疾走感及强烈中毒性。擅长使用韃靼战法*6。
16、なの厨兄贵
BBクッキー☆剧场作者，是少数提供RU素材且以RU為主角的剧场作者。
超展开兼大量下流画面是其特色（杂○及屎尿成份注意）。
17、biim兄贵
主要投稿RTA（实机游戏竞速）影片的作者。早期也有製作BB先辈剧场。
形象是ゆっくり妖梦。
其RTA影片採用称為BIIM SYSTEM的面板。

这面板对观眾理解十分有帮助，引来许多RTA影片作者模仿，风潮甚至传至KOTY的总评影片。
18、かずちんP
别称「底辺ボカロPの末裔」，如字面所述前身為VOCALOID曲作者但是作品都仅仅四位数再生因此堕落改行製作クッキー☆MAD后反而跃升成為有名クッキー☆MAD作者之一，再生数更是原本作V家歌的数百倍。
淫梦化后仍然有持续发表新的原创曲，不过即使有淫梦民的加持仍然无法突破五位再生，反而是副业クッキー☆MAD人气扶摇直上，甚至成為佔据2015年クッキー☆排行榜前几名的霸主。
クッキー☆MAD的特色是擅用运用本业所锻鍊出来的技术（声源调教、PV/MMD等）。
毕竟现实环境随便合成一个ほのぼの神社アレンジ都比费力作V家原创曲还比较多再生数也是不争的事实
底边ボカロP特有的排行榜集计初期投稿意识（集计初期投稿通常在当期排行榜动画中能取得较高积分，因此成為底边ボカロP衝高作品曝光度的手段之一）
